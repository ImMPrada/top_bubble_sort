### How to try

- clone the repo
- got to de repo's folder on your machine
- this repo solution is using Ruby 3.1.2, if you are using rbenv: `rbenv local 3.1.2`
- run `bundle install`, this allows you to run specs
- solution propoused is at lib/
- to run specs: `rspec`
